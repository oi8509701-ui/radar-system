📐 Fractal H-Resonator Architecture

The antenna uses a self-similar H-fractal geometry that provides multi-band operation through recursive scaling.

LevelFrequencyBar LengthApplication
L010.5 GHz9.16 mm✓ Main operating frequency
L121.0 GHz4.58 mm2nd harmonic
L242.0 GHz2.29 mm3rd harmonic
L384.0 GHz1.15 mm4th harmonic
Scaling Factor: Each level is scaled by 1/√2 (≈0.707) from the previous level, creating a self-similar fractal structure.
