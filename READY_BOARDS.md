# 📡 AERIS-10N Phased Array Radar - Готовые платы

## ✅ ГОТОВЫЕ К ПРОИЗВОДСТВУ

### 1. BACKPLANE_16T (Backplane для 16 тайлов)

**Файл:** `Fractal_32x16/BACKPLANE_16T_v1.kicad_pcb`  
**Статус:** ✅ **ГОТОВА**  
**Назначение:** Материнская плата для подключения 16 тайлов антенной решётки

#### 📋 Характеристики:
- **Разъёмы:** 16× коннекторов для тайлов
- **Сигналы:** REF, SPI, SYNC, RESET, +3V3, GND
- **Слоёв:** 2 (F.Cu, B.Cu)
- **Толщина:** 1.6 мм
- **Размеры:** ~300×200 мм

#### 🔌 Интерфейс к каждому тайлу:
| Pin | Сигнал | Описание |
|-----|--------|----------|
| 1-2 | GND | Земля |
| 3 | +3V3_DIG | Питание 3.3V |
| 4 | CLK_REF | Clock reference |
| 5-7 | SPI | SCK, MOSI, MISO |
| 8 | CS | Chip Select |
| 9 | SYNC | Глобальная синхронизация |
| 10 | RESET | Сброс ADAR1000 |

---

### 2. PowerAmplifierBoard (Усилитель мощности)

**Файл:** `PowerAmplifierBoard/RF_PA.kicad_pcb`  
**Статус:** ✅ **ГОТОВА**  
**Назначение:** Плата усилителей мощности для TX тракта радара

#### 📋 Характеристики:
- **Усилители:** QPA2962 GaN (10W каждый)
- **Частота:** 10.5 GHz
- **Мощность:** до 10W на канал
- **Слоёв:** 2 (F.Cu, B.Cu)
- **Толщина:** 0.762 мм (Rogers RO4350B)

#### 🔧 Компоненты:
- **PA:** QPA2962 (GaN усилители)
- **Разъёмы:** SMA для RF входа/выхода
- **Питание:** +28V для GaN усилителей
- **Охлаждение:** Тепловые переходы для отвода тепла
