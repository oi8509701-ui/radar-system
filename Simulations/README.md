## 🎯 Simulation Types

### 1. Array Pattern Simulation

**Purpose:** Calculate radiation pattern of fractal H-tree array

**Files:**
- `Array_Pattern/antenna_array_fractal.m` (NEW - for fractal H-tree)
- `Array_Pattern/Antenna_array.m` (Original - reference)

**What it simulates:**
- Array Factor (AF)
- Element Pattern (EP)
- Total Pattern = AF × EP
- E-plane and H-plane cuts
- 2D heatmap visualization

**For Fractal H-Tree:**
- 4×4 array (16 elements, 4 channels)
- 8×16 array (128 elements, 16 channels)
- H-tree feed network phase relationships

---

### 2. H-Tree Feed Network Simulation

**Purpose:** Analyze phase and amplitude balance of H-tree feed

**Files:**
- `Fractal_HTree/htree_phase_balance.m` (NEW)
- `Fractal_HTree/htree_amplitude_balance.m` (NEW)

**What it simulates:**
- Phase delay from center to each element
- Amplitude loss from center to each element
- Path length matching
- Impedance transformations

**Key Metrics:**
- Phase error: <5° RMS
- Amplitude error: <0.5 dB
- Return loss: <-15 dB @ 10.5 GHz

---

### 3. Fractal Dimension Analysis

**Purpose:** Calculate fractal dimension and scaling properties

**Files:**
- `Fractal_HTree/fractal_dimension.m` (NEW)

**What it calculates:**
- Fractal dimension (D)
- Scaling factor (1/√2 per iteration)
- Number of elements per iteration
- Total path length per iteration
